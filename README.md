# real-estate-dashboard
 Real Estate Listing Dashboard with React, Shadcn, and Vite.

## Run Locally

Go to the project directory

```bash
  cd react-estate-dashboard
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
